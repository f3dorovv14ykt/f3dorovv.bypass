## 🛠 Технологический стек

Проект базируется на нескольких ключевых модулях, которые работают полностью автономно:

1.  **Data Ingestion**: Сбор данных из проверенных OpenSource источников через Bash-скрипты.
2.  **Multi-threaded Validation**: Система проверки на базе `nc`. Мы используем **30 параллельных потоков** и таймаут в 3 секунды, чтобы отсеять 99% мусора.
3.  **Smart Header Generation**: Каждый файл подписки получает заголовок с актуальным временем Якутска и данными о трафике.
4.  **Automatic Delivery**: Обновление происходит каждые 3 часа, гарантируя доступ к свежим прокси.
