### 🛠 Как это работает?
1. **GitHub Actions** каждые 3 часа проверяет источники из папки `data/`.
2. Скрипт фильтрует только живые и рабочие серверы.
3. Результаты разбиваются по 200 штук и сохраняются в папку `configs/`.
