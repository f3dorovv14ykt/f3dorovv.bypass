## 🛠 Как устроена эта магия?

Проект работает на базе мощного конвейера, который не знает усталости:

1.  **Авто-сбор (Infection)**: Скрипты обходят десятки источников и выкачивают тысячи VLESS-конфигов.
2.  **Глубокая очистка (Validation)**: Мы не просто копируем ссылки. Система использует многопоточную проверку через `nc` (Netcat) с таймаутом в 3 секунды. Если сервер тормозит — он летит в корзину.
3.  **Smart Formatting**: Живые узлы упаковываются в файлы по 200 штук и снабжаются метаданными подписки (Userinfo).
4.  **Time-Sync**: Каждое обновление помечается реальным временем **Якутска (UTC+9)**.
